更换了我的博客主题

2016-12-17

今天尝试着去更换自己博客的主题。本来是想自己在Hexo默认的Landscape主题基础上进行修改的，以为用简单的CSS样式，修改’blockquote’格式及字体等简单设置即可，但是在尝试了一下不行。

于是，还是老老实实线实用比人已经做好的Theme吧。知乎上搜索一番，觉得’yilia’主题挺好的，可以满足在电脑端和移动端的较好表现，同时字体、引用格式及分享都算是比较满意。

简单过程如下：

安装

  1. 首先在git bash中，CD到Hexo根目录下；
  2. 通过以下命令进行安装yilia主题

$ git clone https:\\github.com/litten/hexo-theme-yilia.git themes/yilia

上述clone的主题地址，可以在该主题github网页上看到。

配置

安装完成之后，需要在Hexo根目录下的_config.yml文件中的theme改为yilia。文件通过notepad++打开。

更新

通过如下命令将主题更新：

cd themes/yilia

git pull

如果，打开自己的博客主题格式任然没有改变的话。可以尝试在Hexo根目录下做一遍如下程序：

hexo clean

hexo g

hexo d

一般情况下，再次打开后格式就会变成新的主题格式。
